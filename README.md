# Animated Sidebar Navigation

A modern sidebar navigation menu built using HTML and CSS.  
This project features a sleek vertical navigation panel with icons, hover effects, and a smooth interactive design.

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Transitions

## Features
- Modern vertical sidebar layout
- Navigation icons with labels
- Hover and active state effects
- Clean UI with glass-style design
- Notification badge indicator
- Responsive and centered layout

## Purpose of the Project
This project was created as part of my front-end development practice to improve my skills in building navigation components, UI layouts, and interactive CSS effects.
